Project Summary: 
This project analyzes two major U.S. transportation datasets: NHTS (travel behavior) and NGSIM (vehicle trajectories). The goal was to create clear visualizations, run an Intelligent Driver Model (IDM) simulation, and provide insights for a transportation planning group.

Tasks Completed: 
Cleaned and visualized key variables from both datasets.
Created a bar chart, histogram, boxplot, and multiple time‑series plots.
Customized at least one plot using matplotlib or seaborn.
Implemented an IDM simulation to model vehicle behavior.
Documented selected variables, methods, findings, and references.
Prepared a Gantt chart, timesheet, annotated code, and written report.

User Guide: 
Install required Python packages (pandas, numpy, matplotlib, seaborn).
Place NHTS and NGSIM CSV files in the data folder.
Run the script or notebook to generate all plots and simulations.
View generated figures in the output folder and refer to the report for interpretation.
